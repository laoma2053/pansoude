# 项目介绍

PanSou 是一个高性能的网盘资源搜索 API 服务，支持 TG（Telegram）频道搜索和自定义插件搜索。系统设计以性能和可扩展性为核心，支持并发搜索、结果智能排序和网盘类型分类。

## 核心特性

- **高性能搜索**：并发执行多个 TG 频道及异步插件搜索，显著提升搜索速度；工作池设计，高效管理并发任务
- **网盘类型分类**：自动识别 14 种网盘链接，按类型归类展示
- **智能排序**：基于插件等级、时间新鲜度和优先关键词的多维度综合排序算法
- **异步插件系统**：支持通过插件扩展搜索来源，支持"尽快响应，持续处理"的异步搜索模式
- **二级缓存**：分片内存 + 分片磁盘缓存机制，大幅提升重复查询速度和并发性能
- **可选认证**：基于 JWT 的无状态认证，默认关闭，私有部署时按需开启

## 适用场景

| 场景 | 说明 |
|------|------|
| 个人网盘搜索站 | 自部署前后端，快速搭建私人搜索工具 |
| 前端接入 API | 仅部署纯 API 版，自行开发搜索界面 |
| 批量资源搜索 | 通过 API 批量搜索网盘资源 |
| 插件二次开发 | 基于插件接口对接任意搜索来源 |

## 项目结构

```
pansou/
├── api/            # HTTP 路由与请求处理层
├── config/         # 配置加载（环境变量）
├── docs/           # 原始设计文档
├── model/          # 数据模型定义
├── plugin/         # 插件系统（70+ 插件）
├── service/        # 核心搜索业务逻辑
├── util/           # 工具库（缓存、连接池、HTTP 工具）
├── main.go         # 程序入口
├── Dockerfile      # Docker 构建文件
└── docker-compose.yml
```
